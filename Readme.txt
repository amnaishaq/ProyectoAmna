## Introducción 
Show majesty, una aplicación para encontrar y explorar películas. La idea es ayudar a las personas a descubrir nuevas películas de manera fácil y divertida. Con Show majesty, puedes buscar películas por género y año o cualquier . También puedes ver las mejores películas de cada año y compartir tus opiniones sobre ellas. La aplicación incluye un carrusel de imágenes para que puedas ver algunas fotos de las películas antes de elegir qué ver.

## Objetivos del Proyecto:

se debe elegir el genero y el año y el numero de peliuculas a mostrar, luego le das al botón y el te mostrara las peliculas, una peliucula puede  pertencer  a varios genros y en tabla pelicula de la api tmdb solo almacena cod genero pero nos interesan los nombres.
